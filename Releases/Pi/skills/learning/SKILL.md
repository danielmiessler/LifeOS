# Learning — Capture & Compound

## Learning Cycle

1. **Experience** — Encounter new information or situation
2. **Extract** — Identify the key insight (one sentence)
3. **Connect** — Link to existing knowledge (What does this relate to?)
4. **Generalize** — Abstract to a reusable principle
5. **Apply** — Use the insight in a new context
6. **Review** — Revisit periodically for retention

## Capture Formats

| Format | When | Length |
|--------|------|--------|
| Insight card | After a discovery | 2-3 sentences |
| Lesson learned | After a mistake | Situation + cause + fix |
| Pattern | Recurring theme | Abstracted rule |
| Mental model | Deep understanding | Framework explanation |
| Question | Open curiosity | Question + current thinking |

## Compounding Rules

1. New insights must link to at least one existing insight
2. Contradictions are flagged — don't merge, debate
3. Every insight gets a confidence rating (1-5)
4. Review unconfident insights first when new evidence appears
5. Outdated insights are archived, not deleted

## Spaced Repetition

- **Day 1**: Initial capture
- **Day 3**: First review — verify accuracy
- **Day 7**: Second review — find new connections
- **Day 30**: Third review — abstract further
- **Day 90**: Archive or integrate into core knowledge
