# Creative Generation — Content Creation

## Domains

### Writing
- **Structure**: Hook → Body → Resolution
- **Voice**: Define tone, POV, audience
- **Drafting**: Skeleton → Flesh → Polish
- **Formats**: Article, essay, fiction, poetry, script, documentation

### Visual Art & Design
- **Input**: Medium, style, color palette, composition
- **Iteration**: Describe changes relative to previous version
- **Constraints**: Format, resolution, style references
- **Prompting**: Subject + style + mood + technical specs

### Music & Audio
- **Structure**: Intro → Verse → Chorus → Bridge → Outro
- **Elements**: Tempo, key, instrumentation, dynamics
- **Formats**: Score, chord progression, arrangement notes

## Iterative Refinement Loop

1. **Generate** initial output from brief
2. **Review** against criteria (originality, coherence, fit)
3. **Identify** specific elements to improve
4. **Revise** with targeted direction
5. **Repeat** until quality threshold met

## Quality Checks

- Does it meet the brief?
- Is the intended audience served?
- Does it have a clear structural arc?
- Is it internally consistent?
- Does it evoke the intended response?
