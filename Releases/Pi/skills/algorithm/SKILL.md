# Algorithm — Structured Problem Solving

## 7 Phases

| Phase | Action | Output |
|-------|--------|--------|
| **OBSERVE** | Gather data without bias | Raw observations |
| **ORIENT** | Analyze context, constraints, patterns | Situation model |
| **DECIDE** | Choose approach from options | Action plan |
| **ACT** | Execute the plan | Results |
| **MONITOR** | Track outcomes against expectations | Deviation data |
| **REFLECT** | Evaluate what worked | Insights |
| **LEARN** | Update mental models | Updated practices |

## Mode Classifier

| Mode | When | Phases Emphasized |
|------|------|-------------------|
| Explore | Unknown territory | OBSERVE, ORIENT |
| Execute | Known solution path | DECIDE, ACT |
| Optimize | Existing process | MONITOR, REFLECT |
| Debug | Failure state | ORIENT, ACT (corrective) |
| Learn | Post-mortem | REFLECT, LEARN |

## Effort Tiers

- **Quick** (5 min): Single pass through all 7 phases, 1-2 sentences each
- **Standard** (15 min): Documented output per phase, 2-3 options in DECIDE
- **Deep** (1 hr+): Multiple iterations, external validation, full audit trail
