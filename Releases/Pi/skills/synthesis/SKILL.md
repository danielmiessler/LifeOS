# Synthesis — Research Synthesis & Reporting

## Process

1. **Collect** — Gather all sources, findings, and raw data
2. **Verify** — Cross-reference claims across sources
3. **Categorize** — Group findings by theme or dimension
4. **Map** — Identify relationships, conflicts, gaps
5. **Structure** — Build logical argument flow
6. **Write** — Produce structured synthesis
7. **Cite** — Attribute every claim to source

## Cross-Source Verification

| Situation | Action |
|-----------|--------|
| Sources agree | Note consensus, high confidence |
| Sources conflict | Present both sides, note evidence weight |
| Single source | Flag as unconfirmed, lower confidence |
| No source | Flag as speculation or opinion |

## Report Structure

1. **Executive Summary** — Key findings in 3-5 bullets
2. **Scope & Method** — What was examined and how
3. **Findings** — Thematic sections with evidence
4. **Analysis** — Patterns, contradictions, implications
5. **Gaps** — What is unknown or uncertain
6. **Conclusions** — Main takeaways
7. **Recommendations** — Actionable next steps
8. **Sources** — Full citation list

## Argument Mapping

- **Claim** → **Evidence** → **Warrant** (why evidence supports claim)
- Flag **missing warrants** (logical leaps)
- Note **counterarguments** and their strength
